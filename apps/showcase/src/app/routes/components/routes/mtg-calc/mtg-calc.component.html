Mortgage Calculator

A configurable mortgage calculator. Currently only a very basic input and amortization schedule output. As features are added make sure everything is configurable to be enabled / disabled.

Installation

HTML

Typescript Config Example: Setting up the mortgage calculator
Settings

autoCalculate false

When true, sets the calculations to auto calculate to update the amortization schedule and pie chart. Also hides the "Calculate" button.

chartOptions - backgroundColor #6c757d, #ff6600

This will allow you to set the background color for the pie chart

chartOptions - hoverBackgroundColor #b4babe, #ffb27f

This will allow you to set the background hover color for the pie chart

interestRate 3.5

Sets the default interest rate for the calculator

loanAmount 350000

Sets the default loan amount for the calculator

termOptions 15 and 30 years

Changes the dropdown options for the terms

terms 30

Changes the default selected term
